# Byte - 학생회 관리 플랫폼

국립한밭대학교 컴퓨터공학과 제 42대 학생회 "Byte" 관리 플랫폼입니다.

## 주요 기능

### 👥 부원 관리

- 부원 정보 조회 및 관리
- 부서별 필터링
- 부원 정보 수정 및 삭제 (회장 전용)

### 📝 게시판

- 게시글 작성, 수정, 삭제
- 카테고리별 분류 (공지, 일반, 회의록)
- 고정 게시글 기능
- 권한 기반 읽기/쓰기 제어
  - 전체 공개
  - 부장 이상만 열람
  - 특정 부서만 열람
  - 작성자만 열람
- 멘션 기능 (@부원이름)
- 첨부파일 업로드
- 리치 텍스트 에디터 (텍스트 포맷팅, 링크, 이미지)

### 📅 캘린더 및 일정 관리

- 월간 캘린더 뷰
- 일정 추가, 수정, 삭제
- 기간 일정 지원 (시작일 ~ 종료일)
- 시간 설정/미설정 옵션
- 게시글과 일정 연결
- 일정 카테고리 (회의, 행사, 일정, 기타)

### 📋 안건 관리

- 안건 등록 및 관리
- 상태 추적 (진행중, 완료, 보류)
- 우선순위 설정 (높음, 보통, 낮음)
- 담당자 지정
- 부서별 분류
- 게시글/일정과 연결

### 👤 프로필 및 설정

- 프로필 정보 조회 및 수정
- 프로필 사진 업로드
- 비밀번호 변경
- 로그아웃

### 🎯 대시보드

- 전체 부원 수
- 오늘의 일정
- 진행중인 안건
- 최근 게시글 목록
- 월간 캘린더 위젯

## 기술 스택

- **Frontend**: Next.js 16, React 19, Styled Components, Framer Motion
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL (Prisma ORM)
- **Deployment**: Vercel
